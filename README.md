# Plugins for ComIn

This is a collection of plugins for ComIn developed in the context of the
WarmWorld-Smarter project ProImpact.

## Plugins included here:

1. `precipitation_accumulation_aes.py`: Plugin used for extracting precipitation accumulated over a user-defined interval (in seconds). Works for AES physics. Corresponding documentation for this plugin can be found [here](docs/precipitation_accumulation_aes.md).
2. `utci.py`: ComIn implmentation of the xclim's calculation of the Universal Thermal Climate Index (UTCI). Works for AES physics. Documentation for this plugin can be found [here](docs/utci.md).
3. `wind_3d`: Extract wind speed at different heights. Works for AES physics. Docs are located [here](docs/wind_3d.md).
